# 8zenith – Human Maybe

**"เราไม่ได้ทำนายอนาคต... เราแค่วิเคราะห์เหตุปัจจัย"**

8zenith คือระบบ **Multi-Engine AI** ที่วิเคราะห์มนุษย์จาก **Bazi, Numerology, Human Design** และศาสตร์อื่น ๆ โดยไม่ "ทำนาย" อนาคต แต่ใช้หลัก **อิทัปปัจจยตา (Idappaccayatā)** — การวิเคราะห์เหตุปัจจัย — เพื่อช่วยให้ผู้ใช้เข้าใจตัวเองและออกแบบชีวิตของตัวเอง

---

## 🧠 ปรัชญาหลัก (Core Philosophy)

| หลักการ | คำอธิบาย |
|---------|----------|
| **Human Maybe** | เราไม่ฟันธง เราไม่ทำนาย เราแค่เสนอ "ความเป็นไปได้" |
| **Idappaccayatā** | วิเคราะห์เหตุปัจจัย ไม่ใช่ทำนายอนาคต |
| **Council of 5** | ระบบทำงานผ่าน 5 Engines ที่สะท้อนจิตวิญญาณมนุษย์ |
| **Card Status Game** | เข้าใจ "เกมสังคม" โดยไม่ตัดสินศีลธรรม |
