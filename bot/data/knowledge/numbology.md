# คัมภีร์เลขศาสตร์ไทย (Thai Numerology)

## 1. ตารางค่าพลังอักษร (Machine Readable JSON)
**กฎเหล็ก:** นี่คือแหล่งข้อมูลเดียวในการแปลงอักษรเป็นตัวเลข ห้ามเดาหรือใช้ความจำ

```json
{
  "1": ["ก", "ด", "ถ", "ท", "ภ", "ฤ", "า", "อุ", "ำ", "่", "ุ"],
  "2": ["ข", "ช", "ง", "บ", "ป", "เ", "แ", "ู", "้"],
  "3": ["ฆ", "ต", "ฑ", "ฒ", "ฃ", "๋"],
  "4": ["ค", "ธ", "ญ", "ร", "ษ", "ฅ", "ะ", "ิ", "โ", "ั"],
  "5": ["ฉ", "ณ", "ฌ", "น", "ม", "ห", "ฎ", "ฬ", "ฮ", "ึ"],
  "6": ["จ", "ล", "ว", "อ", "ใ"],
  "7": ["ซ", "ศ", "ส", "ี", "ื", "๊"],
  "8": ["ย", "พ", "ฟ", "ผ", "ฝ", "็"],
  "9": ["ฏ", "ฐ", "ไ", "์"]
}
```

## 2. วิธีการคำนวณ (The Map-Reduce Algorithm)
**จงทำตามขั้นตอนต่อไปนี้อย่างเคร่งครัด ห้ามข้าม ห้ามประมาณค่าในใจ**

1.  **แตกชื่อ (MAP):** แยกชื่อและนามสกุลออกเป็นตัวอักษรทีละ 1 ตัว (รวมสระ วรรณยุกต์ และตัวการันต์ด้วย)
2.  **ค้นหาค่า (LOOKUP):** สำหรับตัวอักษรแต่ละตัว ให้ค้นหาว่าอยู่ในกลุ่มเลขใดในตาราง JSON ข้างบน
3.  **เขียนสมการ (FORMULA):** เขียนสมการแสดงผลรวม เช่น `"ว(6) + ร(4) + ก(1) + ฤ(1) + ช(2)"` โดยแสดงค่าของทุกตัวอักษร
4.  **รวมผล (REDUCE):** บวกเลขทั้งหมดในสมการให้ได้ผลลัพธ์
5.  **ตรวจทาน (VERIFY):** นับจำนวนตัวอักษรในชื่อเท่ากับจำนวนค่าในสมการหรือไม่

## 3. ตัวอย่างการคำนวณ (Few-Shot Examples)

### ตัวอย่าง 1: ชื่อ "วรกฤช"
- ตัวอักษร: ว, ร, ก, ฤ, ช
- ค้นหาค่า:  
  ว อยู่ในกลุ่ม 6 → 6  
  ร อยู่ในกลุ่ม 4 → 4  
  ก อยู่ในกลุ่ม 1 → 1  
  ฤ อยู่ในกลุ่ม 1 → 1  
  ช อยู่ในกลุ่ม 2 → 2  
- สมการ: 6 + 4 + 1 + 1 + 2
- ผลรวม: 14

### ตัวอย่าง 2: นามสกุล "สุนทรธรรมนิติ"
- ตัวอักษร: ส, ุ, น, ท, ร, ธ, ร, ร, ม, น, ิ, ต, ิ
- ค้นหาค่า:  
  ส = 7, ุ = 1, น = 5, ท = 1, ร = 4, ธ = 4, ร = 4, ร = 4, ม = 5, น = 5, ิ = 4, ต = 3, ิ = 4
- สมการ: 7+1+5+1+4+4+4+4+5+5+4+3+4
- ผลรวม: 51

### ตัวอย่าง 3: ชื่อ "นิธิตยา"
- ตัวอักษร: น, ิ, ธ, ิ, ต, ย, า
- ค้นหาค่า:  
  น=5, ิ=4, ธ=4, ิ=4, ต=3, ย=8, า=1
- สมการ: 5+4+4+4+3+8+1
- ผลรวม: 29

### ตัวอย่าง 4: ชื่อ "สมชาย" (สำหรับฝึกเพิ่ม)
- ตัวอักษร: ส, ม, ช, า, ย
- ค้นหาค่า:  
  ส=7, ม=5, ช=2, า=1, ย=8
- สมการ: 7+5+2+1+8
- ผลรวม: 23

## 4. ข้อควรระวัง
- **ห้าม** ใช้การจำว่าชื่อนี้ควรได้เลขอะไร ให้ทำตามขั้นตอนเสมอ
- **ห้าม** ข้ามตัวอักษร แม้จะเป็นสระหรือวรรณยุกต์ที่ดูเหมือนไม่มีเสียง
- **ห้าม** ใช้ตารางอื่นนอกเหนือจากตารางในข้อ 1
- **ห้าม** คิดในใจ ให้เขียนสมการออกมาให้เห็นเสมอ
